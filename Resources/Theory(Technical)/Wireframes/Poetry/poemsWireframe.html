Poetry Homepage wireframe

The poems are categorized into Themes. Each theme is a respective panel. This was chosen to allow some sort of ordering to the poems rather than an arbitary long list. Once the user hovers over a panel the webpage changes colour respectively. Example: on the Love section the page turns red. This was added in to create an atmosphere and more interactivity to the page to aid UX. The use of panels add a more distinct structure to the page, this improves the UI from the first design.
