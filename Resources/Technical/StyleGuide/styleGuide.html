Website style guide

UI design

The UI design is aimed to create a casual yet professional feel. The layout of the poems and gallery achieves this via a grid system. The grid allows content to feel more collected.

Colour scheme

The colour pallette for the website is aimed to create a professional atmosphere. The use of a biege background (#F6FFF8) is aimed to create a subdued and book-like feeling. The red (#B23850) is used for the headings.

CSS structures

For the Gallery and Poetry pages the CSS grid structure was used. This was chosen as the grid structure will allow the page to be responsive (it will auto adjust its coloums and rows based on the current pages scaling). Furthermore, the grid structure allows for a more controlled layout.
