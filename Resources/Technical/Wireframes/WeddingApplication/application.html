Arranged wedding application wireframe

The respective input elements were chosen with the UI/UX principles in mind. For the name field a standard textbox is used. For the Age a slider is used. This was chosen to prevent invlid data input as the range of the value can be set and it ensures that the user can only provide a numeric value. With UX in mind, a slider can give the user intsant feedback and it comfortable to use. For the Dhal, Cultured and Alcohol fields, checkboxes are used as the input is either true or false. A checkbox ensures that the user can only enter a true or false value.It is also easy to use and natural feedback to the user aiding UX (tick for true).
