Poetry Homepage wireframe

The poems are categorized into Themes. Each theme is a respective panel. This was chosen to allow some sort of ordering to the poems rather than an arbitary long list. The use of panels add a more distinct structure to the page, this improves the UI from the first design.

The page follows a CSS grid layout. This was chosen to allow the page to be more accessible (panels will automatically adjust based on the pages scaling). Each panel displays the poems on a list structure, aiding UX and visual feedback.
