The uses of Metadata - what, why and how.

Metadata is essentially extra information about existing data[1]. With regards to HTML, this information will not be displayed on the webpage but is machine parsable (can be read/ interpreted by a computer). Metadata is used specifically in HTML to specify keywords, authors, last modified details and page descriptions. This data is used by browsers, search engines and general web services.

Metadata is implemented via the "meta" tag and must always be within the element. The element is used to provide information about the webpage, and it is here that sets website accessibility and search engine optimization (SEO)[2]. SEO is vital to web developers and clients as they require their website to be easily found by customers and within the first few results of the search engine. If metadata characteristics are used effectively this will ease the workings of bots and make the website more accessible.
